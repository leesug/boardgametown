## 🎨 방법 1: Canva로 만들기 (초보자 추천!)

### Step 1: Canva 접속
1. https://www.canva.com 접속
2. 무료 회원가입
3. "디자인 만들기" 클릭

### Step 2: Open Graph 이미지 만들기
```
1. "맞춤 크기" 선택
2. 가로: 1200, 세로: 630 입력
3. "새 디자인 만들기" 클릭

4. 배경 설정:
   - 좌측 메뉴에서 "요소" 클릭
   - "그라데이션" 검색
   - 보라색/파란색 그라데이션 선택

5. 게임 제목 추가:
   - 좌측 메뉴에서 "텍스트" 클릭
   - "제목 추가" 선택
   - "그레이트 킹덤" 입력
   - 폰트 크기: 80-100
   - 색상: 흰색
   - 가운데 정렬

6. 부제 추가:
   - "부제 추가" 선택
   - "이세돌 9단 디자인 전략 보드게임" 입력
   - 폰트 크기: 40-50
   - 색상: 흰색

7. 아이콘 추가 (선택사항):
   - "요소" → "성" 또는 "왕관" 검색
   - 적당한 아이콘 배치

8. 다운로드:
   - 우측 상단 "공유" → "다운로드"
   - 파일 형식: JPG
   - 파일명: great-kingdom-og.jpg
```

### Step 3: Twitter 이미지 만들기
```
1. 기존 디자인 복제
2. 크기 조정: 1200 x 675
3. 레이아웃 약간 조정
4. 다운로드: great-kingdom-twitter.jpg
```

### Step 4: 파비콘 만들기
```
1. 새 디자인: 512 x 512 (정사각형!)
2. 간단한 로고/아이콘만 배치
3. 다운로드: PNG 형식

4. 온라인 변환기 사용:
   - https://favicon.io/favicon-converter/ 접속
   - 512x512 이미지 업로드
   - 자동으로 모든 크기 생성
   - 다운로드한 파일들을 프로젝트 폴더에 복사
```

## 🎨 방법 2: Photoshop/GIMP (고급 사용자)

### Open Graph 이미지 만들기:
```
1. 새 파일: 1200 x 630 픽셀, 72 DPI
2. 그라데이션 배경 적용
3. 게임 스크린샷 배치 (투명도 50%)
4. 텍스트 레이어 추가
   - 제목: 굵은 글꼴, 80pt
   - 부제: 보통 글꼴, 40pt
5. 그림자 효과 추가 (가독성)
6. JPG로 내보내기 (품질: 85%)
```

### 파비콘 만들기:
```
1. 새 파일: 512 x 512 픽셀
2. 간단한 아이콘 디자인 (복잡하면 안 보임!)
3. PNG로 저장
4. https://realfavicongenerator.net/ 에서 변환
```

## 🎯 디자인 팁

### 색상 선택:
```
그레이트 킹덤 추천 색상:
- 주 색상: #667eea (보라빛 파랑)
- 보조 색상: #764ba2 (진한 보라)
- 강조 색상: #ff4757 (빨강)
- 텍스트: #ffffff (흰색)
```

### 폰트 선택:
```
제목: 
- Noto Sans KR (한글)
- Montserrat (영문)
- 굵기: Bold 또는 ExtraBold

본문:
- Noto Sans KR (한글)
- Open Sans (영문)
- 굵기: Regular 또는 Medium
```

### 레이아웃 팁:
1. **제목은 상단 1/3 지점**에 배치
2. **텍스트 주변에 여백** 충분히
3. **대비가 높은 색상** 사용 (가독성)
4. **중요한 내용은 중앙**에
5. **그림자 효과**로 텍스트 강조
