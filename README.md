## 🆘 자주 묻는 질문 (FAQ)

### Q1: 이미지 없이도 작동하나요?
**A:** 네! 이미지 없이도 모든 SEO 태그가 작동합니다. 
단, 소셜 미디어 공유 시 텍스트만 표시됩니다.

### Q2: 효과가 언제부터 나타나나요?
**A:** 
- 소셜 미디어 공유: 즉시
- 구글 검색: 1-2주 후
- 검색 순위 상승: 3-6개월

### Q3: 다른 페이지도 최적화해야 하나요?
**A:** 네! great-kingdom.html 등 다른 페이지도 
같은 방식으로 최적화하면 더 좋습니다.

### Q4: Netlify에 어떻게 배포하나요?
**A:** 
1. Netlify 사이트에 로그인
2. "Sites" → 해당 사이트 선택
3. "Deploys" → "Upload" 버튼
4. 파일들을 드래그 앤 드롭

### Q5: Google Search Console이 뭔가요?
**A:** 구글이 제공하는 무료 SEO 도구입니다.
- 검색 성능 확인
- 색인 상태 확인
- 문제점 발견
→ https://search.google.com/search-console

## 🎓 SEO 용어 사전

- **SEO**: Search Engine Optimization (검색엔진 최적화)
- **Meta Tag**: 페이지 정보를 담은 HTML 태그
- **Open Graph**: 소셜 미디어 공유 프로토콜
- **Schema.org**: 구조화된 데이터 표준
- **Canonical URL**: 대표 URL (중복 방지)
- **Sitemap**: 사이트 구조 지도
- **robots.txt**: 크롤러 규칙 파일
- **Rich Snippet**: 검색 결과 강화 표시
- **CTR**: Click Through Rate (클릭률)
