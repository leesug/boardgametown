# 🎨 아줄 (Azul) 게임 개발 가이드

## 📖 게임 규칙 (간소화 버전)

### 게임 목표
포르투갈 타일 장인이 되어 궁전 벽을 아름답게 장식하고 가장 많은 점수를 획득하세요!

### 게임 구성
- **타일**: 5가지 색상 (파랑, 노랑, 빨강, 검정, 흰색)
- **공장 디스플레이**: 공용 타일 선택 공간 (2인: 5개, 3인: 7개, 4인: 9개)
- **플레이어 보드**: 각자의 패턴 라인 + 벽 + 바닥 라인
- **패턴 라인**: 5줄 (1칸, 2칸, 3칸, 4칸, 5칸)
- **벽**: 5x5 그리드 (각 행마다 다른 색상 순서)
- **바닥 라인**: 벌점 공간 (최대 7칸)

### 게임 흐름

#### 1단계: 공장 채우기
- 각 공장에 주머니에서 타일 4개씩 무작위 배치
- 중앙에는 아무것도 없음 (1라운드 시작 마커만)

#### 2단계: 타일 선택 (턴제)
플레이어는 차례대로:
1. **공장 또는 중앙에서 같은 색 타일 전부 가져가기**
   - 공장에서: 선택한 색 외 나머지는 중앙으로
   - 중앙에서: 모든 같은 색 타일 가져가기
   - 중앙에서 처음 가져가는 사람은 "시작 마커(-1점)" 획득

2. **자기 패턴 라인 중 한 줄에 배치**
   - 해당 줄이 수용할 수 있는 만큼만 배치
   - 넘치는 타일은 바닥 라인으로
   - 같은 색만 한 줄에 놓을 수 있음
   - 이미 벽에 있는 색은 그 줄에 배치 불가

3. **모든 공장과 중앙이 비면 3단계로**

#### 3단계: 벽 타일링 (라운드 종료)
각 플레이어:
1. **완성된 패턴 라인의 타일 1개를 벽으로 이동**
   - 나머지는 버림
2. **점수 계산**
   - 기본: 1점
   - 가로로 인접한 타일마다 +1점
   - 세로로 인접한 타일마다 +1점
3. **바닥 라인 벌점 계산** (-1, -1, -2, -2, -2, -3, -3)
4. **시작 마커를 가진 사람이 다음 라운드 시작 플레이어**

#### 게임 종료
- 누군가 벽의 가로줄 1개를 완성하면 게임 종료
- 추가 점수:
  - 가로줄 완성: 줄당 +2점
  - 세로줄 완성: 줄당 +7점
  - 같은 색 5개 완성: 색당 +10점
